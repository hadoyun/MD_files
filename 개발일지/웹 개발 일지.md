## div 태그

div 태그는 항상 100% 너비를 차지 한다...!


**시멘틱 태그 붙이기!**

```html
    <div id = "header"> <!-- 과거 -->
        <h1>
        <a href='#'>헤이버</a>
        </h1> 
        <h2>검색창</h2>
            <fieldset>
            <legend>검색</legend>
                <input />
                <button>검색</button>
            </fieldset>
    </div>
```

